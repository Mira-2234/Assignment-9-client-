# 🐾 Pet Adoption Platform (Client)

## 📌 Project Name
Pet Adoption Platform

## 🎯 Purpose
A full-stack MERN application that allows users to browse pets, submit adoption requests, and manage pet listings in a secure and user-friendly system.

---

## 🚀 Live URL
https://assignment-9-client-mu.vercel.app/

---

## ✨ Features

- 🐶 Browse all available pets
- 🔍 Search, filter, and sort pets
- 📄 View detailed pet information
- ❤️ Submit adoption requests (authenticated users only)
- 🧾 Manage personal adoption requests
- 🏠 Add, edit, and delete pet listings (owners only)
- 📊 Dashboard with listings and request management
- 🔐 JWT authentication with secure routes
- 📱 Fully responsive design (mobile, tablet, desktop)
- 🔔 Toast notifications instead of alerts

---
